# Tech Wiki

A comprehensive wiki for all tech stuff.

## 🌙 Features

- **Modern Dark Theme** - Easy on the eyes with a sleek, GitHub-inspired dark color scheme
- **Wiki-Style Navigation** - Sidebar table of contents organized by tech stack
- **Full-Text Search** - Quickly find what you're looking for
- **Responsive Design** - Works great on desktop and mobile

## 📚 Topics

- **Copilot CLI** - Complete setup guide for GitHub Copilot CLI on Windows & macOS

## 🛠️ Tech Stack

Built with:
- [Jekyll](https://jekyllrb.com/) - Static site generator
- [Just the Docs](https://just-the-docs.github.io/just-the-docs/) - Documentation theme
- [GitHub Pages](https://pages.github.com/) - Hosting

## 🚀 Local Development

```bash
cd docs
bundle install
bundle exec jekyll serve
```

Then open [http://localhost:4000](http://localhost:4000)
