# GitHub Copilot CLI
{: .fs-9 }

AI-powered command line assistance for developers.
{: .fs-6 .fw-300 }

---

## Overview

GitHub Copilot CLI brings the power of AI assistance directly to your terminal. It helps you:

- Generate shell commands from natural language descriptions
- Explain complex commands
- Fix command errors
- Integrate with corporate knowledge bases through Work IQ

## Quick Start

{: .highlight }
> Already have everything installed? Jump straight to the [Setup Guide](setup) for step-by-step instructions.

### Prerequisites at a Glance

- Git
- Node.js (optional, if using npm install method)
- Azure CLI (for corporate authentication)

### Installation Methods

| Platform | Command |
|:---------|:--------|
| Windows (winget) | `winget install GitHub.Copilot` |
| macOS (Homebrew) | `brew install copilot-cli` |
| npm (all platforms) | `npm install -g @github/copilot` |
